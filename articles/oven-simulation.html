oven simulation

Illustrates a model that can be used by other models. Also illustrates use of a forcefield to simulate a simple continuous variable.

a model .

the interface

  • heatTo a number
  • isDoorOpen a boolean
  • (maybe) roomTemp a number (external stream)
  • currentTemp a number

the implementation

  • a forcefield
  • a weak forceX at heatTo
  • (maybe) when door is open, a weak forceX at roomTemp
  • ticker hasInterval (250)
  • forcefield listensTo ticker
  • forcefield contains oventemp
  • currentTemp listensTo forcefield, maps with pluck oventemp, x
